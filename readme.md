# 🧙‍♂️ Sistema de Heróis em JavaScript

Um projeto simples em JavaScript para demonstrar conceitos de Programação Orientada a Objetos (POO), como classes, métodos e lógica condicional.

---

## 📖 Sobre o projeto

Este projeto cria uma classe chamada `Heroi`, onde cada herói possui:

- Nome  
- Idade  
- Tipo (classe)

Cada tipo de herói possui um ataque diferente, definido através de uma estrutura `switch`.

---

## ⚙️ Tecnologias utilizadas

- JavaScript 
- Node.js
